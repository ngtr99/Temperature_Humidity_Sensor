This project involves the development of an IoT-based temperature and humidity monitoring and prediction system using an ESP32 microcontroller and Arduino framework for real-time data acquisition. Environmental data was collected through sensors (e.g., DHT11) and processed using Arduino (C/C++), then integrated with a web-based interface for visualization and analysis. On the software side, Python programming was used for data preprocessing, analysis, and implementing basic machine learning models for predicting temperature and humidity trends. The system leverages multiple tools and technologies, including ESP32 development kit, Arduino IDE, I2C communication (SSD1306 OLED display), Serial communication for debugging, web technologies (HTML, CSS, JavaScript), and cloud deployment platforms (e.g., Vercel). This project demonstrates hands-on experience in embedded systems, IoT architecture, data pipelines, and machine learning integration, highlighting the ability to bridge hardware and software for real-time environmental monitoring and predictive analytics.
